##### Experiment 3: Slideshow



###### Aim



* To develop an Android application that displays a slideshow of images automatically at fixed intervals. 



###### Algorithm



* Start Android Studio and create a new Android project.



* Add an ImageView in the layout file.



* Place multiple images in the drawable folder.



* Create an array of image resources in MainActivity.



* Declare a Handler object to control timing.



* Create a Runnable method to change images.



* Set the image to ImageView using setImageResource().



* Increase the image index after every change.



* Reset the index when it reaches the end of the array.



* Use handler.postDelayed() to change images every few seconds.



* Run the application to view the slideshow.
